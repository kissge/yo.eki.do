yum --enablerepo=remi-php56 install php-pecl-mongo

とかで解決。（これでいいんか）

よくよく考えたら /etc/yum.repos.d/* を適切に編集して必要なレポジトリをenabledにするのがよい。何もわかっていなかった。 (2015-10-24)
